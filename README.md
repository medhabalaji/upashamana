# upashamana
Pain relief made easy through vibration therapy.

# what it does
Upashamana is a portable and accessible pain relief device that utilises coin vibration motors and vibration therapy, in conjunction with an esp32 microcontroller, and TP4056 motor drivers.

# why it's useful
Chronic pain is a $7+ global proble. There is a notable gap in modern pain relief in providing portable and affordable solutions. This is where Upashamana comes in. Using coin vibration motors and a microcontroller, it provides a compact, deep, drug-free relief to individuals with chronic pain. It is flexible, discreet, and works anywhere on the body.
